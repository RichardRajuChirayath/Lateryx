How It Works

From Code to Confidence

01

Graph Construction

Lateryx maps your infrastructure as a directed graph. Resources become nodes, permissions become edges.

02

Pathfinding

Using NetworkX algorithms, we find all paths from Internet to ProtectedData.

03

Change Analysis

When a PR is opened, we compare the "before" and "after" graphs to detect new or shortened paths.

04

Breach Detection

If your change creates a highway where there used to be a wall, Lateryx flags it as a Causality Breach.
